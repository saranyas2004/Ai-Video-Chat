# 🤖 AI Voice Companion

An AI-powered voice assistant that detects your face and starts a conversation to help improve English speaking skills.

## 🚀 Features
- 👁️ Face Detection using OpenCV
- 🎤 Voice Input (Speech Recognition)
- 🔊 Voice Output (Text-to-Speech)
- 🧠 AI Conversation (API-based)
- ⚡ Multi-threading for smooth performance

## 🛠️ Tech Stack
- Python
- OpenCV
- SpeechRecognition
- pyttsx3
- Requests API

## 🎯 Purpose
This project helps users:
- Improve English speaking skills
- Reduce loneliness through conversation
- Practice real-time communication

## ⚠️ Note
Add your own API key before running the project.

## ▶️ How to Run
```bash
pip install -r requirements.txt
python voice_face_app.py
